# Trading App

This is a trading app built with React and Vite.

## Features

- Trading UI
- Market data
- Order book
- Trades
- Positions
- Account

## Development

- `npm install`
- `npm run dev`
- Open [http://localhost:3000](http://localhost:3000) in your browser

## Building

- `npm run build`
- `npm run preview`

## Technologies

- React
- Vite
- Tailwind CSS
- TypeScript
- ESLint
- Prettier
